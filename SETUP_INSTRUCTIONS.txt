Setup Instructions for Interviewers

PREREQUISITES:
--------------
1. Python 3.8 or higher installed
   - Download from: https://www.python.org/downloads/
   - During installation, CHECK "Add Python to PATH"

2. Google Gemini API Key
   - Get one at: https://ai.google.dev/
   - You'll need this for the app to work


INSTALLATION STEPS:
-------------------

Step 1: Configure API Key
   a) Open the file: config.py
   b) Find the line: GEMINI_API_KEY = "YOUR_GEMINI_API_KEY_HERE"
   c) Replace YOUR_GEMINI_API_KEY_HERE with your actual API key
   d) Save the file

Step 2: Install Dependencies
   a) Double-click: setup.bat
   b) Wait for installation to complete (1-2 minutes)
   c) Press any key when prompted
   d) You should see "Setup complete!" message


RUNNING THE APPLICATION:
------------------------

Step 1: Start the App
   a) Double-click: run.bat
   b) A command window will open
   c) Wait for "Running on http://0.0.0.0:5000" message
   d) DO NOT close this window

Step 2: Open in Browser
   a) Open your web browser
   b) Go to: http://localhost:5000
   c) You should see the Creative Automation Pipeline interface


USING THE APPLICATION:
----------------------

1. Review Campaign Briefs (Middle Panel)
   - Default briefs for Adobe Firefly and Photoshop are loaded
   - You can edit the JSON if desired

2. Add Reference Images (Optional)
   - Place brand assets in the "InputAssets" folder
   - They'll appear in the right panel
   - Use checkboxes to include/exclude them

3. Generate Campaigns

TROUBLESHOOTING:
----------------

Problem: "python is not recognized"
Solution: Install Python and make sure "Add to PATH" was checked

Problem: "Invalid API Key" error
Solution: Check config.py has your correct Gemini API key

Problem: "Module not found" error
Solution: Run setup.bat again

Problem: Port 5000 already in use
Solution: Close other apps using port 5000, or modify app.py (last line)

Problem: Images not generating
Solution: 
   - Check internet connection
   - Verify API key is correct
   - Check Gemini API quota/limits
